Análisis de Llamadas al 144: Línea de Emergencia por Violencia de Género en Argentina
Descripción
Este proyecto se enfoca en el análisis de los datos recopilados de las llamadas realizadas al 144, la línea de emergencia por violencia de género en Argentina. El propósito principal es profundizar en el entendimiento de las tendencias, patrones y características de estas llamadas para obtener una visión más completa de la dinámica de la violencia de género en el país.

Además de cumplir con este objetivo, este proyecto tiene una dimensión educativa y de desarrollo de habilidades. A través del análisis de estos datos, se busca practicar y demostrar conocimientos en el área de análisis de datos, incluyendo técnicas de limpieza, visualización y modelado de datos.

Instalación:

Para poder tener acceso al proyecto y ejecutarlo correctamente, es necesario tener instalado lo siguiente:

* Ambiente virtual:
    - En este caso, he trabajado con Visual Studio Code version 1.89.1.
    - Visual Studio Code: Disponible en [Visual Studio Code](https://code.visualstudio.com/).
    - Puedes ingresar este comando para verificar la versión que tenes de VSC:
  
                ---Bash------
                code --version
                -------------

* Python:
    - La version utilizada en este caso fue 3.12.3.
    - su descarga está disponible en [Python.org](https://www.python.org/).
    - Comando para verificar la versión:
                ---Bash------
                python --version
                -------------

* Librerías utilizadas:
  - Pandas:
    - Versión utilizada de Pandas: 2.2.2
    - Comando para instalar en macOS y Linux:
                ---Bash------
                pip install pandas
                -------------
    - Comando para instalar en Windows:
                ---Bash------
                pip install pandas
                -------------
  - NumPy:
    - Versión de pandas: 2.2.2
    - Comando para instalar en macOS, Linux y Windows:
                ---Bash------
                pip install NumPy
                -------------

  - Matplotlib:
    - Versión de Matplotlib: 3.8.3.
    - Comando para instalar en macOS, Linux y Windows:
                ---Bash------
                pip install matplotlib
                -------------
    
  - Seaborn:
    - Version: 0.13.2.
    - Comando para instalar en macOS, Linux y Windows:
                ---Bash------
                pip install seaborn
                -------------
  - Plotly:
    - Versión: 5.20.0.
    - Comando para instalar en macOS, Linux y Windows:
                ---Bash------
                pip install plotly
                -------------
 
  - scikit-learn:
    - Versión: 1.4.2. 
    - Comando para instalar en macOS, Linux y Windows:
                ---Bash------
                pip install scikit-learn
                -------------

  - tabulate: 
    - Versión: 0.9.0
    - Comando para instalar en macOS, Linux y Windows:
                ---Bash------
                pip install tabulate
                -------------

Estos comandos instalarán las bibliotecas necesarias y sus dependencias para ejecutar el proyecto. Si ya tienes instaladas algunas de estas bibliotecas pero necesitas actualizarlas a las últimas versiones, puedes utilizar el comando `pip install --upgrade` seguido del nombre de la biblioteca que deseas actualizar.
